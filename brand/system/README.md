# Melanoir Brand System
**v1.1 · 2026-05-28**

멜라누아 브랜드 문서 시스템. 각 문서는 독립된 역할을 가지며, 모두 Brand Platform에서 파생된다.

---

## 문서 구조

| 파일 | 역할 | 독자 | 변경 빈도 |
|---|---|---|---|
| `01_brand_platform.md` | 브랜드 헌법 — 핵심 DNA | 내부 전략, 투자자, 파트너 | 거의 변경 없음 |
| `02_brand_identity.md` | 보이스·비주얼·타이포·로고 기준 | 디자이너, 카피라이터, 에이전시 | Brand Platform 변경 시 |
| `03_content_ops_manual.md` | 콘텐츠 제작 플레이북 | 외주 에디터, 내부 마케터 | 운영 경험 누적 시 |
| `04_embo_product_brief.md` | Embo PMU 잉크 제품 브리프 | 마케터, 유통 파트너, 시술자 교육 | 제품 업데이트 시 |
| `05_employer_brand.md` | 채용·팀 아이덴티티 | 채용 후보자, 채용 채널 | 채용 상황 변화 시 |
| `06_photoshield_brief.md` | Photoshield Core 원료 브리프 | 화장품 제조사, B2B 파트너 | 제품 업데이트 시 |
