# Melanoir Embo — 샘플 테스트 프로젝트 플랜 v1.3

**작성일**: 2026-05-29
**프로젝트 목적**: 리뉴얼 Embo 제품의 시술 결과 검증 + 판매 채널 개설용 콘텐츠 수집
**담당**: Chief Architect (official@melanoir.co.kr)

---

## 프로젝트 개요

| 항목 | 내용 |
|---|---|
| 샘플 수량 | 15개 (시술자 1인당 3회분) |
| 모집 규모 | 5명 |
| 모집 구성 | 기존 네트워크 2~3명 + 공개모집 2~3명 |
| 예상 기간 | 약 8주 (샘플 발송 기준) + 장기 팔로업 |
| 소통 채널 | 구글 폼(단계별 3개) + 이메일(official@melanoir.co.kr) |
| 산출물 | 시술 전후 사진 + 전문가 평가 데이터 → 스마트스토어 상세페이지 소재 |

> **샘플 3회분 구성**
> - 1회분: 패드(실리콘) 테스트 — 물성·핸들링 감각 확인
> - 2회분: 실제 시술
> - 3회분: 리터치 (필요 시)

> **샘플 발송 형태**
> 완제품 패키지가 아닌 샘플 형태로 발송 (지퍼백 또는 파우치 + 안내 카드).
> 완제품 언박싱 경험은 정식 출시 시로 보류.

### 고객(피시술자) 장기 데이터 수집 구조 (정식 출시 후)

## VIP 공급가 재판매 방지 전략

### 구조적 억제 장치

**보증서 일련번호 추적**
개별 제품에 고유 번호 부여. 시술자 → 고객 전달 경로가 정품인증으로 추적됨.
인증 안 된 제품은 고객 리워드 미지급 → 고객이 정품인증을 할 유인이 생기며 자연스럽게 유통 경로 확인 가능.

**시술자 계정 단위 월 구매 한도**
VIP 등급은 시술자 계정에 귀속. 월 구매 수량 기준(예: 5개 이상) 초과 시 사유 확인 프로세스 적용.
개인사업자 시술자의 실제 소비량을 벗어나면 자동으로 이상 신호.

**주문-배송 구조 유지**
재고를 쌓아두는 구조가 아니므로 대량 매입 후 재판매 유인 자체가 낮음.

**VIP 조건 명시**
VIP 할인은 "본인 시술에 한함"을 약관에 명시. 위반 시 등급 취소 조항 포함.
