## 1. 브랜드 표현 원칙 (제작 전 필독)

### 허용 표현 기준
- **Data-First**: 수치가 먼저 말하고, 설명이 뒤를 잇는다. `"97%다. 이것은 세포가 살아남은 비율이다."` 순서.
- **선언형 종결**: 모든 문장은 `"~다."`로 끝난다. 의문형·청유형 금지.
- **맥락 제시, 비교 주장 금지**: 직접 비교 우위 표현 불가. 사실 나열 후 판단은 독자에게.
  - ✅ `"시중 PMU 잉크의 일반적인 판매 단위는 다회 사용이다. 멜라누아 엠보는 단 1회분으로 설계된다."`
  - ❌ `"타 제품보다 안전하다"` / `"업계 최초"` / `"최고의 성분"`

### 절대 금지 표현
| 금지 표현 | 이유 |
|---|---|
| "최고의", "혁신적인", "최초의" | 비교 우위 주장 |
| "안전하다" (단독 사용) | 맥락 없는 주장 |
| "친환경이다", "무독성" | 검증 방식 불명확 |
| "화학적 부채(Chemical Debt)" | 헤드라인·핵심 주장 전면 금지 (본문 단일 언급 허용) |
| EWG "인증" | NGO 자체 평가임 — "EWG 1등급으로 평가됨"만 허용 |
| "가장 ~한" | 최상급 표현 금지 |
| 1,500개/mm² | 검증 불가 수치 — 사용 금지 |

### 공식 수치 (출처 명기 필수)
| 수치 | 표기 형식 | 출처 |
|---|---|---|
| 피부 자극 지수 | **0.00** | ISO 10993-23:2021 |
| 세포 생존률 | **97%** | ISO 10993-5 |

### SECTION 01 — Hero Declaration
**목적**: 3초 안에 브랜드 가치를 각인한다.

**헤드라인 (확정)**
```
내 몸이 잘 아는 색소, 멜라닌.
```

**서브텍스트 방향**
- 부연 설명 없이 헤드라인만으로 섹션을 완성하는 것이 가장 강하다.
- 필요 시 제품명과 용량 표기만 추가: `Melanoir Embo · 0.5g · 단회 사용`

**비주얼 지시사항**
- 배경: 순백(#FFFFFF) 또는 절대 블랙(#000000)
- 제품 실물 사진: 매트 질감의 패키지. 스튜디오 조명. CG 금지.
- 텍스트와 이미지 비율: 여백 60% 이상 유지.

### SECTION 02 — The Single Ingredient
**목적**: 단일 색소 원칙을 브랜드의 핵심 철학으로 제시한다.

**헤드라인 방향**
```
색소는 하나다. 유멜라닌.
```

**본문 방향**
- 멜라누아 엠보에는 단 하나의 색소 성분이 존재한다. 유멜라닌.
- 일반 PMU 잉크의 일반적인 구성은 목표 색조를 위한 복수의 합성 색소 혼합이다. 멜라누아 엠보는 이 구조를 채택하지 않았다.
- 성분 간 상호작용 변수가 없다. 시험 데이터가 해당 성분에 온전히 귀속된다.
- 유멜라닌의 발색 범위는 브라운-블랙이다. 피부가 자체적으로 생성하는 색의 범위와 일치한다.

**비주얼 지시사항**
- 성분명 `Eumelanin` 또는 `유멜라닌` 단어를 크게 타이포그래픽 처리.
- 성분 목록(단 하나)을 미니멀하게 세로 나열.
- 현미경 또는 분자 구조 실사 이미지 활용 가능 (CG 제외).

### SECTION 03 — The Numbers
**목적**: 3개의 핵심 수치로 신뢰를 확립한다. 이 섹션이 상세페이지 전체의 신뢰 중심축이다.

**구성**
3개 수치를 각각 독립적인 카드 또는 타이포그래픽 요소로 배치한다.

```
0.00
피부 자극 지수
ISO 10993-23:2021 기준 — 측정 가능한 범위의 최솟값

97%
세포 생존률
ISO 10993-5 기준 — 비독성 판정 기준(70%) 대비 27%p 상회

EWG 1등급
EWG 자체 평가 기준으로 평가됨
```

**카피 방향**
- 수치 먼저, 설명이 뒤를 잇는다.
- `0.00`은 단순히 낮은 수치가 아니다. 비자극 범주의 최솟값이다. — 이 맥락을 부기한다.
- 원본 성적서는 페이지 하단 또는 별도 링크로 제공한다. 이 섹션에서는 수치만 전달한다.

**비주얼 지시사항**
- 수치를 가장 크고 얇은 폰트로 표현. 숫자 자체가 비주얼이다.
- 배경 무채색(웜그레이 또는 Absolute Black).
- 성적서 원본 캡처 삽입 금지 — 수치 인포그래픽으로 재해석.

### SECTION 04 — Biocompatibility
**목적**: "생체적합성"이라는 핵심 셀링 포인트를 시각적으로 설명한다.

**헤드라인 방향**
```
피부가 이미 알고 있는 분자.
```

**본문 방향**
- 유멜라닌은 피부가 자체적으로 생성하는 색소다. 인체는 이 분자를 이물질로 인식하는 경로가 단순하다.
- 면역 인식 경로의 단순화는 시술 후 회복 변수를 줄인다.
- 세포독성 시험(ISO 10993-5) 결과, 세포 생존률 97%가 이를 수치로 뒷받침한다.

**비주얼 지시사항**
- 인포그래픽 방식: 어려운 화학식 대신 단순한 다이어그램.
- "외부 물질 → 면역 반응" vs "유멜라닌 → 단순 인식 경로" — 두 흐름을 텍스트+라인으로 표현.
- 의료기기 브로셔 수준의 정제된 레이아웃 지향.

### SECTION 07 — Single-Use Philosophy
**목적**: 0.5g 단회 사용 원칙과 가격 근거를 하나의 선언으로 전달한다.

**헤드라인 방향**
```
0.5g.
단 한 번의 시술을 위해 설계된 용량.
```

**본문 방향**
- 반영구화장 잉크는 진피층에 투입되어 수년간 피부에 잔류한다.
- 이 사실을 소재 설계의 출발점으로 삼을 때, 적용해야 할 기준은 일반 화장품이 아닌 의료용 필러에 준하는 수준이다.
- 개봉 후 재사용은 위생과 오염 방지 원칙에 반한다. 멜라누아 엠보는 단회 사용 포장을 원칙으로 한다.
- 279,000원. 이 숫자는 수년간 피부에 지속되는 소재에 대한 책임의 값이다.

**비주얼 지시사항**
- 단상자 + 0.5g 앰플 제품 클로즈업. 고화질 스튜디오 촬영.
- 용량 수치 `0.5g`을 타이포그래픽 포인트 요소로 활용.
- 가격 279,000원은 단독 강조가 아닌, 단회 설계 맥락과 함께 제시.

### SECTION 10 — FAQ
**목적**: 구매 직전의 핵심 질문 4개를 명료하게 해소한다.

**질문 초안 (확정 전 내부 검토 필요)**

```
Q1. 기존 잔흔 위에도 시술이 가능한가?
A. [기술적 사실에 기반한 서술 필요 — 내부 확인 후 작성]

Q2. 유지 기간은 어떻게 되나?
A. 유지 기간은 개인의 피부 대사, 시술 깊이, 생활 환경에 따라 달라진다.
   유멜라닌은 시간이 지나면서 피부의 자연 색소 범위 안에서 흐려진다.
   잔흔의 색 변이가 아닌, 자연스러운 감소가 특성이다.

Q3. 단회 사용 이후 남은 잉크는 어떻게 처리하나?
A. 멜라누아 엠보는 단 1회 시술분으로 설계된 용량이다.
   개봉 후 재사용은 위생 원칙에 반하므로 폐기한다.

Q4. EWG 1등급이라는 의미는 무엇인가?
A. EWG(Environmental Working Group)는 성분 안전성을 연구하는 비영리 민간 기관이다.
   멜라누아 엠보는 EWG의 자체 평가 기준으로 1등급으로 평가됐다.
   이것은 비정부 기관의 자체 평가이며, 법적 인증이 아니다.
```
