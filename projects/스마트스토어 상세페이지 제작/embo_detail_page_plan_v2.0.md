## 1. 브랜드 표현 원칙 (제작 전 필독)

### 허용 표현 기준
- **Data-First**: 수치가 먼저 말하고, 설명이 뒤를 잇는다. `"97%다. 이것은 세포가 살아남은 비율이다."` 순서.
- **선언형 종결**: 모든 문장은 `"~다."`로 끝난다. 의문형·청유형 금지.
- **맥락 제시, 비교 주장 금지**: 직접 비교 우위 표현 불가. 사실 나열 후 판단은 독자에게.
  - ✅ `"시중 PMU 잉크의 일반적인 판매 단위는 다회 사용이다. 멜라누아 엠보는 단 1회분으로 설계된다."`
  - ❌ `"타 제품보다 안전하다"` / `"업계 최초"` / `"최고의 성분"`

### 절대 금지 표현
| 금지 표현 | 이유 |
|---|---|
| "최고의", "혁신적인", "최초의" | 비교 우위 주장 |
| "안전하다" (단독 사용) | 맥락 없는 주장 |
| "친환경이다", "무독성" | 검증 방식 불명확 |
| "화학적 부채(Chemical Debt)" | 헤드라인·핵심 주장 전면 금지 (본문 단일 언급 허용) |
| EWG "인증" | NGO 자체 평가임 — "EWG 1등급으로 평가됨"만 허용 |
| "가장 ~한" | 최상급 표현 금지 |
| 1,500개/mm² | 검증 불가 수치 — 사용 금지 |

## 2. 디자인 시스템 명세

기존 embo.html 레퍼런스에서 추출한 레이아웃 컴포넌트 시스템. 스마트스토어 버전도 이 구조를 기준으로 제작한다.

### 2-1. 레이아웃 그리드
- **최대 너비**: 1120px, 좌우 패딩 40px
- **섹션 여백**: 상하 180px (모바일: 100px)
- **섹션 구분선**: `1px solid #f5f5f7` (얇고 거의 보이지 않는)
- **교대 배경**: 흰 섹션(`#ffffff`) ↔ 연회색 섹션(`#fbfbfd`) 교대 반복

### 2-2. 지그재그 2-컬럼 시스템
```
[섹션 홀수] 이미지(左) | 텍스트(右)
[섹션 짝수] 텍스트(左) | 이미지(右)
```
- 컬럼 비율 1:1, 간격 100px
- 모바일: 단일 컬럼으로 전환, 이미지가 상단에 오도록 역전

### 2-3. 타이포그래피 위계
| 태그 | 역할 | 스타일 |
|---|---|---|
| `h3` | 섹션 레이블 | uppercase, letter-spacing: 0.15em, 회색(#86868b), 1.55rem |
| `h2` | 메인 헤드라인 | bold, 3.95rem, 자간 좁게 |
| `p` | 본문 | 1.65rem, line-height: 1.8, #424245 |
| Hero Title | 임팩트 타이틀 | 7.2rem, weight: 900 |

### 2-4. 컴포넌트 스펙

**미디어 박스 (Media Box)**
- 배경: `#f5f5f7`, border-radius: 40px, height: 550px (고정)
- 이미지/비디오: object-fit: cover
- 오버레이 캡션: 좌하단, 반투명 흰 배경(blur), 라운드 pill 형태
- 캡션 내용: 사진에 대한 팩트 레이블 (예: "전자현미경 관찰 멜라닌 입자 100,000X")

**Stats Bar (수치 바)**
- 위치: Hero 섹션 하단
- 구성: 3~4개 수치 항목 수평 배치, 구분선으로 분리
- 수치 폰트: 3.35rem bold
- 라벨 폰트: 1.25rem, 회색

**Data Grid (데이터 테이블)**
- 구성: 좌측 항목명(bold) / 우측 값(강조색)
- 구분선: `1px solid #d2d2d7`
- ⚠️ 기존 HTML의 강조색 `#0071e3`(애플 블루)은 브랜드 원칙 미부합 → 스마트스토어 버전에서 `#1d1d1f`(블랙) 또는 브랜드 컬러로 교체 필요

**Bar Chart (막대 차트)**
- 컨테이너: `#f5f5f7` 배경, border-radius: 40px
- 막대: 대조군(회색) vs 멜라누아(블랙)
- 수치를 막대 위에 배치, 하단에 캡션
