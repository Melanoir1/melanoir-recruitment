How It Works

작동 방식.

멜라누아 제품을 사용한 시술을 등록하면 자동으로 집계됩니다.

01
제품 사용

멜라누아 제품으로 고객 시술을 진행합니다.

02
시술 등록

정품 보증서의 QR을 스캔하여 시술 내용을 등록합니다. 시술 당일 처리됩니다.

03
등급 자동 산정

최근 6개월 등록 횟수가 기준을 충족하는 순간 등급이 즉시 전환됩니다.
