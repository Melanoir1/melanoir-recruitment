COSMETICS

화장품에 여러 기능성을 동시에 구현합니다.

피부에 닿는 성분은 기능을 가져야 합니다. 멜라닌을 정교하게 기능화해, 기존 화장품 성분이 채우지 못했던 광보호와 항산화의 자리를 채웁니다.

OVERVIEW

개요

피부에 닿는 화장품 성분은 무해한 데서 그치지 않고 능동적으로 기능해야 합니다. 멜라누아는 생체가 빛과 산화로부터 자신을 지키기 위해 진화시킨 멜라닌을 화장품 원료로 재해석합니다. 자외선·블루라이트를 흡수하고 활성산소에 대응하는 멜라닌의 특성을 정교하게 기능화해, 기존 화장품 성분이 채우지 못했던 광보호와 항산화의 자리를 채웁니다.

MECHANISM

작용 메커니즘

멜라닌은 넓은 파장대의 빛을 흡수하는 색소 고분자로, 자외선(UV)뿐 아니라 블루라이트 영역까지 흡광 성능을 보입니다. 동시에 활성산소(ROS)를 소거하고 중금속을 흡착하는 특성을 통해 산화 스트레스에 대응합니다. 멜라누아는 천연유래 멜라닌을 정제·기능화해 제형에 적용 가능한 형태로 가공하며, 불필요한 성분을 덜어낸 단순한 조성을 지향합니다.

DATA

실험 데이터

포토쉴드 코어(Photoshield Core) 원료는 동일 농도 조건의 비교 시험에서 블루라이트 차단 4.5x(vs 산화아연), 비타민 C 대비 지속 항산화 3.8x를 확인했고, 피부 임상 저자극 시험에서 자극 지수 52% 감소를 기록했습니다. 수치는 원료 단독 특성이며, 자세한 데이터와 면책 사항은 Photoshield Core 제품 페이지에서 확인할 수 있습니다.

PRODUCTS

관련 제품

멜라닌 기반 광보호·항산화 원료는 멜라누아의 Photoshield Core로 공급되고 있습니다.
