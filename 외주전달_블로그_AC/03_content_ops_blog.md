## 0. Tone 4원칙 — 블로그 교육 콘텐츠 적용 가이드

> **→ 참조:** Tone 4원칙의 원본 정의는 `01_brand_identity.md` **A-1**이다.
> 이 섹션은 해당 원칙이 **블로그 A/C 교육 콘텐츠에서 어떻게 구현되는지**를 정의한다.
> 원칙의 본질은 고정이며, 구현 방식만 맥락에 맞게 적용된다.

| 원칙 | 본질 (고정) | 블로그 교육 콘텐츠 구현 방식 |
|---|---|---|
| **Data-First** | 수치·메커니즘이 주어, 철학이 술어 | 본문 첫 단락을 수치 또는 검증된 사실로 시작한다. DOI 인용 필수. 마케팅 카피보다 더 엄격하게 적용한다. |
| **Subtraction** | 핵심에 집중, 불필요한 것 제거 | 토픽당 핵심 논점 하나를 유지한다. 단, 그 논점을 충분히 전개하는 깊이는 허용한다. |
| **Declaration** | 선동 없이 단정 짓는다 | **소제목:** 검색 쿼리 매칭을 위한 의문형 허용. **본문·클로징:** 선언형 유지. 청유형·감탄형 금지. |
| **Non-competitive** | 새로운 범주를 정의한다 | 기존 소재의 사실(데이터)을 제시하고 판단은 독자에게 남긴다. 제품명·브랜드 직접 비교 금지. |

**블로그 추가 고정값**
- 본문·클로징 종결: 선언형 ("~다")
- 첫 단락: 수치 또는 핵심 사실로 시작
- 소제목: 의문형 허용 → 본문에서 반드시 선언형으로 답할 것
- 금지: 청유형 ("~해보세요"), 감탄형, 과도한 수사

## 1. 콘텐츠 레이어 구분

멜라누아 콘텐츠는 목적·법적 포지션에 따라 3개 레이어로 구분한다.
**레이어 간 혼용은 광고법 위반 리스크를 수반한다.**

| 레이어 | 섹션 코드 | 적용 범위 | 핵심 원칙 |
|---|---|---|---|
| **교육 레이어** | A / C | 멜라닌 과학 교육, 산업·시장 교육 | 성분 사실만. 제품명·효능 주장 없음. Mission 언어 부재. 단독 채널 운영 가능. |
| **제품 레이어** | B / D | 브랜드·제품 콘텐츠, 소비자 의사결정 지원 | 제품 시험 수치 허용. ISO 성적서 근거 필수. Luxury 포지셔닝 적용. A/C 채널과 혼용 불가. |
| **브랜드 선언 레이어** | — | 랜딩페이지, IR, 채용, 공식 브랜드 발행물 | Mission/Vision + Unrivaled Tech 전면 적용. 창업 철학 언어 허용. |

### 레이어 간 혼용 금지

- 교육(A/C) 채널에서 제품 수치·제품명을 전면에 배치하면 성분 과학 교육 지위가 소멸된다.
- 교육 콘텐츠 말미 허용 최대 범위: "멜라누아는 이러한 멜라닌을 적용한 소재 플랫폼이다."
- Mission/Vision 언어는 브랜드 선언 레이어에만 등장한다. 제품 레이어에 삽입하지 않는다.

### Unrivaled Tech 필러의 레이어별 적용

| 레이어 | 적용 방식 |
|---|---|
| 교육 레이어 | 적용 없음 |
| 제품 레이어 | ISO 수치로 기술 독보성 간접 서술 |
| 브랜드 선언 레이어 | "멜라누아만이 할 수 있는 것"으로 직접 선언 |

---


> **※ 외주 전달용:** 블로그 A/C 섹션 전용 편집본. §2 Luxury 언어화 정의(제품 레이어 전용)는 제외됨.

## 3. 광고법 준수 원칙

### 성분 과학 진술 vs. 제품 효능 주장

| 구분 | 정의 | 허용 채널 |
|---|---|---|
| **성분 과학 진술** | 멜라닌 물질의 화학적·생물학적 특성을 논문·표준 기반으로 기술 | 교육 레이어 (A/C) |
| **제품 효능 주장** | 멜라누아 엠보(특정 제품)가 특정 효과를 낸다고 주장 | 제품 레이어 (B/D) — 시험 성적서 근거 필수 |

경계 사례:
- ✅ "유멜라닌은 Internal Conversion을 통해 흡수된 UV 에너지를 열로 전환한다." → 성분 사실 (A 가능)
- ❌ "멜라누아 엠보가 UV 차단 효과를 제공한다." → 제품 효능 주장 (B, 근거 필수)
- ✅ "ISO 10993-23 시험에서 피부 자극 지수 0.00이 측정되었다." → 시험 결과 (B 가능)
- ❌ "멜라누아 엠보는 피부에 안전하다." → 법적 정의 없는 표현 (금지)

### 법적 정의 없는 단어 금지 및 대체표

| 금지 단어 | 대체 표현 |
|---|---|
| 안전하다 | "면역 반응 없음" / "비자극 범주의 최솟값" / "ISO 10993 기준 통과" |
| 무독성이다 | "세포 생존률 97% — ISO 10993-5 비독성 기준(70%) 상회" |
| 효과적이다 | "피코초 단위로 작동하는 Internal Conversion 반응이 확인된다" |
| 친환경이다 | "효소에 의한 생물학적 분해 경로가 알려져 있다" |
| 검증되었다 (단독) | "ISO 10993-5 시험에서 [수치]가 측정되었다" |

### 수치 인용 기준

1. **출처 필수**: DOI 또는 공인 표준 번호 첨부
2. **미확인 수치**: ⚠️ 표시 + 보수적 표현으로 조정
3. **범위 표현**: 단일 연구 수치를 일반화하지 않음 — "연구에서 보고된 바 있다" 수준으로 제한
4. **비교 수치**: 출처 없는 "업계 평균" 류의 수치 사용 금지

### A/C 섹션 제품명 언급 허용 범위

- ✅ "멜라누아는 이러한 멜라닌을 적용한 소재 플랫폼이다."
- ✅ "멜라누아 엠보는 유멜라닌을 PMU 잉크 소재로 구현한 제품이다." (효능 주장 없이)
- ❌ "멜라누아 엠보를 사용하면 이 모든 이점을 경험할 수 있다." → 효능 주장으로 전환

## 4. 채널 변환 가이드

콘텐츠 소스 파일의 5-레이어 스캐폴드를 채널별로 변환하는 기준.

| 채널 | 소스에서 가져올 것 | 버릴 것 | 길이 기준 |
|---|---|---|---|
| **블로그** | 본문 서술 전체 + 검증 데이터 인용 | META 내부 정보 | 1,200–1,800자 |
| **인스타그램 카드뉴스** | 핵심 주장 + 검증 데이터 수치만 | 본문 서술 대부분 | 슬라이드당 40–60자 |
| **유튜브 쇼츠 스크립트** | 핵심 주장 + 본문 서술 앞 2문단 | 상세 메커니즘 | 60초 ≈ 150–180자 |
| **뉴스레터** | 본문 서술 + 표현 주의사항 반영 | 데이터 DOI 상세 | 600–800자 |

**브랜드 보이스 공통 원칙 (채널 공통 고정값)**

> **→ 참조:** `brand/system/02_brand_identity.md` **A-1** (4원칙) 및 **A-3** (금지·권장 단어).
> 채널이 달라도 이 원칙은 변하지 않는다.

## 5. 외주업체 전달용 프롬프트 골격

외주 라이터 또는 AI에게 콘텐츠 소스 작성을 의뢰할 때 사용하는 기본 프롬프트.

```
당신은 멜라닌 과학 콘텐츠의 원본 소스를 작성하는 전문 라이터입니다.
주어진 토픽에 대해 아래 5개 섹션 구조로 작성하세요.

[작성 원칙]
1. 주장하지 않는다. 메커니즘으로 말한다.
2. 수치는 문장 앞에 온다. 해석은 수치 뒤에 온다.
3. 본문은 오해 소환 → 교정 → 메커니즘 설명 → 의미 재정의 순서로 전개한다.
4. 비교 우위를 직접 서술하지 않는다.
   기존 소재의 사실(IARC 분류, 잔류 기간 등)을 수치로 제시하고 판단은 독자에게 남긴다.
5. "안전하다", "뛰어나다", "효과적이다"는 쓰지 않는다.
   "면역 반응 없음", "비자극 범주의 최솟값", "분해 경로 확인"으로 말한다.
6. 모든 수치는 DOI 또는 공인 표준 번호를 첨부한다.
   확인되지 않은 수치는 "⚠️ 미확인" 표시와 함께 표현을 보수적으로 조정한다.
7. 종결 문장은 선언형으로 닫는다. 의문문, 청유형 금지.

[섹션 구조]
META / 핵심 주장 / 본문 서술 (400–600자) / 검증 데이터 (DOI 포함) / 표현 주의사항 (✅/❌)

[광고법 원칙]
A 섹션: 멜라닌 성분의 과학적 사실만 기술. 제품명·제품 수치 언급 없음.
B 섹션: 제품 시험 결과 수치 사용 가능. ISO 인증 기관 성적서 근거 명시 필수.
```

## 6. 콘텐츠 소스 5-레이어 스캐폴드

각 토픽의 표준 구조. 외주업체 제작 및 내부 검수 기준.

| 레이어 | 역할 | 기준 |
|---|---|---|
| **[META]** | 코드·섹션·타겟·채널·우선순위·광고법 포지션 | 내부용. 콘텐츠에 노출 금지. |
| **[핵심 주장]** | 단 하나의 명제. 선언문 3줄 이내. | 수치 포함 권장. |
| **[본문 서술]** | 오해 소환 → 교정 → 메커니즘 → 의미 재정의 | 400–600자. |
| **[검증 데이터]** | DOI 단위 출처 + 수치 | 본문과 독립적으로 검증 가능하게 분리. |
| **[표현 주의사항]** | ✅ 권장 / ❌ 금지 | 브랜드 보이스 + 광고법 기준 + 수치 한계 명시. |

## 7. 팩트체크 레퍼런스 테이블

콘텐츠 작성·검수 시 반드시 대조할 기술 기준값.

| 항목 | 올바른 정보 |
|---|---|
| 멜라노사이트 밀도 | **955 cells/mm²** (전 연령 가중 평균) — "1,500"은 오류 |
| 피부 자극 시험 표준 | **ISO 10993-23:2021** (Irritation) |
| 피부 감작 시험 표준 | **ISO 10993-10:2021** (Sensitization) — 별개 표준 |
| 세포독성 시험 표준 | **ISO 10993-5** / 비독성 기준: 세포 생존률 70% 이상 |
| IARC Group 2B 정의 | 인간 근거 제한적 **OR** 동물 충분한 근거 (OR 조건) |
| EWG 멜라닌 등급 | **1등급** — NGO 자체 평가, 법적 인증 아님 |
| Embo 피부 자극 지수 | **0.00** (ISO 10993-23 기준) |
| Embo 세포 생존률 | **97%** (ISO 10993-5 기준) |
| Internal Conversion | 초기 ~100 펨토초, 이완 ~4 피코초 |
| EU REACH 규제 | Commission Regulation (EU) 2020/2081 / 발효 2022-01-04 |
